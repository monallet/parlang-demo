# Authority and Release Boundary

Parlang is owned and controlled by the Parlang project owner.

This repository may be published as a public demo because it contains only rough UI concepts, static sample data, and non-confidential descriptions. It must not be treated as the production application source.

Do not add:

- Production Supabase URLs, keys, service-role keys, database connection strings, SMTP credentials, or GitHub tokens
- The private iOS/macOS application source
- Generated dictionary databases or unreleased vocabulary data
- Production release plans, private backend schemas, or unpublished study algorithms

Any expansion of this repository beyond the demo scope requires explicit approval from the project owner.
