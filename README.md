# Parlang Demo

The project *Parallel/Comparative Language Learning*, namely [**Parlang**](https://parlang.org), stands for the concept of multilingual-comparative language study. The companion and present of the other languages, either acquired or desired, would principally help the learner acquire the target language/languages with solid memorisation, dynamic mental association, and linguistic comparison across vocabularies and grammars in different languages. The **Parlang2** as the iOS/macOS application will firstly be released upon passes of beta tests and customer reviews. The web-app comes along with the official release.

Parlang Demo is a public, simplified SwiftUI preview of the Parlang/Parlang2 app concept, specifically for the pre-release customer reviews and authorized usages. It is not the production iOS/macOS application and does not include any confidential source, private backend logic, cloud configuration, full dictionaries, or release-only study flows.

## Demo Scope

- `Menu`: language selection for the visible vocabulary surface
- `Lists`: vocabulary list browsing with static sample entries
- `Mine`: explanation of the study session concept and a brief mention of personalised study sessions
- `Settings`: search filters only; other settings intentionally remain placeholders

## Running

Open the package in Xcode:

```bash
open Package.swift
```

Select the `ParlangDemo` executable target and run on a supported iOS or macOS destination.

## Confidentiality Boundary

The production Parlang app remains confidential until the project owner releases it through the App Store or gives explicit written approval for a broader source release. This repository is authorized only as a lightweight public demo.
